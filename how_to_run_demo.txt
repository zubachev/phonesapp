Как запустить демку
1. Загружаем ноутбук Diplom.ipynb из папки Week_7_Final (*python 3, pickle и flask должны быть предварительно установлены)
2. Запускаем на выполнение код из ноутбука
3. Проходим по ссылке http://127.0.0.1:5000/ (пример)
4. В текстовом редакторе запущенной веб-страницы пишем, например, положительный отзыв (реально крутой смартфон! :))
5. Нажимаем "Оценить". Получаем результат: 'позитивный'

Примечание: в Week_7_Final должна находиться директория templates, в которой должен находиться файл index.html (Week_7_Final/templates/index.html). Кроме того в Week_7_Final должны находиться следующие файлы:
1. sentiment_classifier.py
2. UnigramUnprocessedVectorizer.pkl
3. LogisticUnigramUnprocessedTextSentiment.pkl
4. Директория static
